# もん助 法人管理画面 — 要件定義書

**文書バージョン:** 1.0  
**作成日:** 2026-04-14  
**対象システム:** もん助 法人管理画面（SPA）  
**本番URL:** https://monsuke-admin.vercel.app  
**リポジトリ:** https://github.com/TsubakiMan/monsuke-admin

---

## 1. システム概要

### 1.1 目的
介護・福祉業界向けワーカーマッチングサービス「もん助」を利用する法人が、自社の事業所・ワーカー・求人・勤怠・請求等を一元管理するための管理画面。

### 1.2 対象ユーザー
もん助と契約している法人の管理者（例：社会福祉法人渓仁会）

### 1.3 技術構成
| 項目 | 内容 |
|------|------|
| 構成 | 単一HTMLファイル（SPA） |
| フロントエンド | HTML / CSS / JavaScript（フレームワーク不使用） |
| フォント | Noto Sans JP（Google Fonts） |
| ホスティング | Vercel |
| ソース管理 | GitHub（TsubakiMan/monsuke-admin） |

### 1.4 デザイン方針
| 項目 | 仕様 |
|------|------|
| ブランドカラー | オレンジ（#E97316） |
| アクセントカラー | ティール（#0D9488） |
| サイドバー | ダーク（#1E293B） |
| レイアウト | 左サイドバー固定 + 右コンテンツ領域 |
| テーブル | ストライプ表示（偶数行に背景色） |
| カード | 上部にアクセントボーダー付き |

---

## 2. 画面一覧・機能定義

### 2.1 ダッシュボード（デフォルト表示）

**目的:** 法人全体の利用状況を一目で把握する。

#### 統計カード（4列）
| カード | 表示内容 |
|--------|----------|
| 今月の利用額 | 金額 + 前月比（%） |
| 稼働中ワーカー | 人数 / 登録総数 + 待機中・休止中の内訳 |
| 今月の稼働件数 | 件数 + 前月比（%） |
| 未読メッセージ | 件数 + メッセージ画面へのリンク |

#### 月別利用額推移チャート
- CSSバーチャートによる直近6ヶ月の推移表示
- **事業所フィルタ:** 法人全体 / 各事業所ごとに切替可能
- 当月のバーはオレンジ（ブランドカラー）でハイライト

#### 事業所別 今月の利用状況（テーブル）
- 各事業所の件数・金額を表示

#### 直近のアクティビティ
- 種類別フィルタ対応（すべて / マッチング / 稼働完了 / レビュー / 請求・支払い / ワーカー / キャンセル）
- 各アクティビティにアイコン・内容・経過時間を表示

---

### 2.2 事業所一覧

**目的:** 法人に属する事業所の一覧を確認し、各事業所の管理画面へ遷移する。

#### 機能
| 機能 | 詳細 |
|------|------|
| 検索 | 事業所名・エリアで検索（スペース区切りOR検索） |
| ステータスフィルタ | 「求人掲載中」「未掲載」で絞り込み |
| ページネーション | 10件ずつ表示 |
| 管理画面ボタン | 各事業所の管理画面へ遷移 |

#### テーブルカラム
事業所ID / 事業所名 / エリア / ステータス / 操作

#### ステータス定義
| ステータス | バッジ色 | 意味 |
|-----------|---------|------|
| 求人掲載中 | 緑 | もん助上で求人を公開している |
| 未掲載 | 赤 | 求人を掲載していない |

#### 制約事項
- 事業所の追加機能は**なし**（もん助側で管理）

---

### 2.3 法人情報

**目的:** 法人の基本情報を確認する。

#### 表示項目
法人ID / 法人名 / 代表者名 / 設立日 / 住所 / 電話番号 / メールアドレス / 契約プラン / 契約開始日

#### 制約事項
- 全フィールド**読み取り専用**（編集ボタンなし）

---

### 2.4 ワーカー管理

**目的:** もん助でマッチングしたことがあるワーカーの一覧を確認する。

#### 統計カード（3列）
| カード | 表示内容 |
|--------|----------|
| マッチング済みワーカー数 | 累計人数 |
| 累計マッチング回数 | 総回数 |
| 平均評価 | ★評価 |

#### 機能
| 機能 | 詳細 |
|------|------|
| 検索 | ワーカー名・IDで検索 |
| 資格フィルタ | 介護福祉士 / 初任者研修 / 実務者研修 / 看護師 / 無資格 |
| 事業所フィルタ | 勤務先事業所で絞り込み |
| ページネーション | 10件ずつ表示 |

#### テーブルカラム
ワーカーID / 名前 / 資格 / 勤務先事業所（複数表示可） / マッチング回数 / 評価 / 詳細ボタン

#### 制約事項
- 稼働状況ステータス（稼働中/待機中/休止中）は**表示しない**
- ワーカーは複数の事業所で勤務した履歴を持つ

---

### 2.5 求人管理

**目的:** 各事業所が掲載している求人の状況を確認する。

#### 統計カード（3列）
公開中の求人数 / 応募総数（今月） / マッチング率

#### テーブルカラム
求人ID / 事業所名 / 職種 / 日時 / 時給 / 応募数 / ステータス

#### ステータス定義
| ステータス | 意味 |
|-----------|------|
| 公開中 | 応募受付中 |
| 締切 | 募集終了 |
| マッチ済 | ワーカーが確定 |

#### 制約事項
- 求人作成機能は**なし**（事業所管理画面側で作成）

---

### 2.6 勤怠管理

**目的:** ワーカーの勤怠記録を確認・エクスポートする。

#### 機能
| 機能 | 詳細 |
|------|------|
| 日付範囲フィルタ | 開始日〜終了日を指定 |
| 事業所フィルタ | 事業所で絞り込み |
| CSVエクスポート | 勤怠データをCSVファイルでダウンロード |
| ページネーション | 10件ずつ表示 |

#### テーブルカラム
日付 / ワーカー名 / 事業所名 / 出勤時間 / 退勤時間 / 実働時間 / ステータス

---

### 2.7 メッセージ

**目的:** ワーカーや事業所からのメッセージを確認する。

#### テーブルカラム
送信者 / 件名（リンク） / 日時 / ステータス（未読/既読）

#### サイドバー表示
- 未読件数をバッジで表示

---

### 2.8 請求・支払い

**目的:** 月次の利用料金と支払い状況を確認し、請求書PDFをダウンロードする。

#### 統計カード（3列）
今月の利用額（税込） / 未払い請求件数 / 年間累計

#### テーブルカラム
月 / 利用件数 / 合計金額（税込） / 支払い期限 / ステータス / 操作

#### 請求書PDF機能
- 各月の行にドロップダウンメニューを配置
- 事業所ごとの請求書PDFを選択してダウンロード可能
- **印刷ベース方式:** 新しいブラウザウィンドウに請求書HTMLを表示し、ブラウザの印刷機能でPDF保存
- 請求書フォーマット: もん助所定のフォーマット（会社情報・明細テーブル・振込先情報を含む）

---

### 2.9 利用明細

**目的:** 過去の利用明細を確認し、CSV/Excelでダウンロードする。

#### テーブルカラム
期間 / 対象事業所数 / 対象人数 / 対象求人数 / 振込関連手数料 / 請求額調整 / 請求額の目安 / 操作

#### ダウンロード機能
- Excelファイル（.xlsx）でのダウンロード対応（SheetJS使用）

---

### 2.10 労働者名簿

**目的:** ワーカーの名簿・支払い情報を期間指定でダウンロードする。

#### 機能
| 機能 | 詳細 |
|------|------|
| 新規追加 | モーダルで期間（開始日〜終了日）を指定して名簿を生成 |
| ダウンロード | Excel (.xlsx) 形式でダウンロード |

#### テーブルカラム
期間 / 合計人数 / 合計金額 / 合計源泉徴収額 / 操作

---

### 2.11 レビュー

**目的:** ワーカーから事業所に寄せられた評価・レビューを確認する。

#### 統計カード（3列）
全体平均スコア / 今月の新着レビュー件数 / 最も高評価の事業所

#### 事業所別平均スコア
- 横棒グラフで事業所ごとのスコアと件数を視覚的に表示
- グラデーション付きバー（オレンジ → ティール）

#### レビュー一覧
| 機能 | 詳細 |
|------|------|
| 事業所フィルタ | 事業所で絞り込み |
| スコアフィルタ | ★5 / ★4 / ★3 / ★2以下 で絞り込み |

#### テーブルカラム
スコア / 事業所名 / ワーカー名 / コメント / 日付

---

### 2.12 レポート・分析

**目的:** 利用状況の推移や事業所別比較を分析する。

#### KPI統計カード（4列）
年間利用総額 / 月平均利用額 / 累計稼働ワーカー数 / 累計稼働回数

#### 月別利用額推移（12ヶ月）
- CSSバーチャートで直近12ヶ月の推移を表示
- 当月バーはオレンジでハイライト

#### 事業所別利用比較（今年度）
- 横棒グラフで事業所ごとの利用額を比較

#### 職種別稼働件数
- 横棒グラフで職種ごとの稼働件数を表示
- 職種: 介護職（日勤）/ 介護職（夜勤）/ 生活支援員 / 看護助手 / 調理補助

#### ワーカー稼働ランキング（Top 10）
- テーブルでワーカーの稼働回数・累計時間・評価を表示
- 上位3名はオレンジでハイライト

---

### 2.13 設定

**目的:** 通知設定と管理者アカウント情報を管理する。

#### 通知設定
- メール通知: 有効 / 無効
- 通知頻度: 即時 / 1日1回 / 週1回

#### 管理者アカウント
- 管理者名・メールアドレスの編集と保存

#### 制約事項
- アカウント削除機能は**なし**

---

## 3. 共通仕様

### 3.1 ナビゲーション構造

```
メイン
  ├── ダッシュボード（デフォルト表示）
  ├── 事業所一覧
  ├── 法人情報
  └── ワーカー管理
運用
  ├── 求人管理
  ├── 勤怠管理
  ├── メッセージ（未読バッジ付き）
  └── レビュー
管理
  ├── 請求・支払い
  ├── 利用明細
  ├── 労働者名簿
  ├── レポート・分析
  └── 設定
```

### 3.2 共通UIコンポーネント
| コンポーネント | 仕様 |
|-------------|------|
| ページネーション | 10件/ページ、前へ/次へボタン + ページ番号 |
| 検索 | インクリメンタルサーチ（入力即時反映） |
| フィルタ | セレクトボックスによる絞り込み（即時反映） |
| ステータスバッジ | 緑(g)/オレンジ(o)/赤(x) の3種類 |
| トースト通知 | 画面下部に一時的なメッセージ表示 |
| モーダル | 背景オーバーレイ付きダイアログ |

### 3.3 廃止済み機能
以下の機能は意図的に実装していない:

| 機能 | 廃止理由 |
|------|---------|
| 求人作成（「＋求人を作成」ボタン） | 事業所管理画面側で管理 |
| 事業所追加（「＋事業所を追加」ボタン） | もん助側で管理 |
| 法人情報編集ボタン | 法人側で直接編集不可 |
| アカウント削除 | 誤操作防止 |
| ワーカー稼働ステータス（稼働中/待機中/休止中） | マッチング履歴ベースの管理に変更 |

## 4. データ仕様

### 4.1 事業所ステータス
| 値 | 表示 | 意味 |
|----|------|------|
| 求人掲載中 | 緑バッジ | もん助上で求人を公開中 |
| 未掲載 | 赤バッジ | 求人未掲載 |

### 4.2 請求書PDF出力
- 出力方式: ブラウザ印刷（window.print）によるPDF保存
- 対応言語: 日本語（CJKフォント対応）
- 含まれる情報: 請求先法人情報、もん助会社情報、明細テーブル（ワーカー名・日付・時間・報酬・交通費・消費税）、振込先銀行情報

### 4.3 Excel出力
- ライブラリ: SheetJS (xlsx)
- 利用明細・労働者名簿のダウンロードに使用

---

## 5. 非機能要件

| 項目 | 仕様 |
|------|------|
| レスポンシブ対応 | モバイル用サイドバートグル（ハンバーガーメニュー） |
| ブラウザ対応 | モダンブラウザ（Chrome, Edge, Safari, Firefox） |
| デプロイ | Vercel（プロダクション自動デプロイ） |
| バージョン管理 | GitHub（masterブランチ） |
