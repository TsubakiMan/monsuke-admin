5. Feature Summary by Page

All 13 pages with key features and constraints

🏠 Dashboard

  • Stats: revenue, active workers, jobs, unread messages
  • Monthly revenue chart (company-wide / per facility)
  • Facility usage breakdown table
  • Activity feed with 6 category filters

🏢 Facilities

  • Text search (OR logic with spaces)
  • Status filter: Posting Active / Not Posting
  • Link to facility admin panel
  • Removed Add facility button

🏛️ Company Info

  • Read-only display of corporate information
  • 9 fields: ID, name, representative, address, etc.
  • Removed Edit button

👥 Worker Management

  • List of all workers who have been matched
  • Shows facilities worked at (multiple) & match count
  • Filters: qualification, facility
  • Removed Work status (Active/Standby)

📋 Job Postings

  • Stats: active postings, applications, match rate
  • Job list table with status badges
  • Removed Create job button

⏰ Attendance

  • Date range & facility filtering
  • Check-in/out times, hours worked
  • CSV export functionality

💬 Messages

  • Message list from workers & facilities
  • Read/unread status management
  • Unread badge on sidebar navigation

⭐ Reviews

  • Visual score bars per facility
  • Filter by facility and score rating
  • Review list table (16 entries)

💰 Billing & Payments

  • Monthly billing overview & payment status
  • Per-facility PDF invoice download
  • Print-based PDF generation method

📄 Usage Details

  • Period-based usage breakdown
  • Excel (.xlsx) download
  • Estimated billing amounts

📒 Worker Registry

  • Generate registry by date range (modal)
  • Excel (.xlsx) download
  • Total headcount, amount, withholding tax

📊 Reports & Analytics

  • 12-month revenue trend chart
  • Facility comparison / job type breakdown
  • Worker performance ranking Top 10

⚙️ Settings

  • Email notification on/off & frequency
  • Admin account info editing
  • Removed Delete account
